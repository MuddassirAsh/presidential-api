Documentation

Below you'll find examples using the Fetch API, but you can use any other http library out there.

Important Notes

  1. Default rate limit is 60 per hour.
  2. Default number of quotes returned from query endpoints is 10.
  3. To view the list of presidents available, click here.

Get a random quote

fetch('https://api.prezidential.xyz/api/random')
      .then(response => response.json())
      .then(quote => console.log(quote))
{
    "quote" : "...",
    "author": "..."
}

Get a random quote by name

fetch('https://api.prezidential.xyz/api/random/kennedy')
      .then(response => response.json())
      .then(quote => console.log(quote))
{
    "quote" : "...",
    "author": "John F. Kennedy"
}

Get 10 random quotes

fetch('https://api.prezidential.xyz/api/quotes')
      .then(response => response.json())
      .then(quote => console.log(quote))
[
    {
        "quote" : "...",
        "author": "..."
    },
    "...9 more"
]

fetch('https://api.prezidential.xyz/api/author/bush')
